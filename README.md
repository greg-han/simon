## Simon &reg;
A remake of the 1980's classic Simon &reg;.

3/22/18: Inital Commit. UI finished! Simply need to code it up and fill in the guts.

Link to site: https://greg-han.github.io/simon/

## Features

* Start: Starts the game.
* Restart: Restarts the game and discards all old patterns.
* Strict: Toggle switch which can toggle "strict mode" on or off. "strict" mode does repeat the current pattern if you miss. Missing will restart the game.
